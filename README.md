# Targetly общая презентация

Интерактивная веб-презентация платформы **Targetly**, построенная на базе Vue 3. 

Презентация рассказывает о ключевых преимуществах платформы: глубокой аналитике, умных алгоритмах закупки трафика (Deep Learning: WIDE, FM, DEEP), абсолютной защите от фрода (анализ TCP-отпечатков) и реальной пользе для бизнеса (оптимизация CPA и ROI).

## Стек технологий

*   **Vue 3** (Composition API, `<script setup>`)
*   **Vite** (сборщик)
*   **PrimeIcons** (векторные иконки)
*   Чистый CSS (анимации, grid-layout, glassmorphism)

## Структура проекта

Презентация состоит из набора независимых Vue-компонентов (слайдов), которые рендерятся и переключаются через главный компонент `SlideDeck.vue`.

*   `src/components/SlideDeck.vue` — Главный контейнер (движок презентации), управляет навигацией, стейтом и переключением слайдов.
*   `src/components/slides/` — Папка со всеми слайдами. Каждый слайд — это отдельный `.vue` файл с инкапсулированной логикой и стилями (некоторые стили вынесены в отдельные `.css` файлы).
*   `public/` — Статические ассеты (картинки, видео, логотипы сетей и партнеров), доступные напрямую.

##  Установка и запуск

1.  Установите зависимости:
    ```bash
    npm install
    ```

2.  Запустите локальный dev-сервер:
    ```bash
    npm run dev
    ```

3.  Сборка для продакшена:
    ```bash
    npm run build
    ```
    (Собранные файлы появятся в папке `dist/`)

## Деплой (текущий флоу)

Проект собирается локально и заливается на сервер через `rsync`. На сервере статику раздает Nginx.

```bash
npm run build && rsync -avz --delete dist/ root@<SERVER_IP>:/var/www/presentation/
```
